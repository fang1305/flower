如何申请机构认证

打开AI学手机客户端并登录：

1.确保您注册时选择的身份为"机构"；

2.点击下方导航栏"我"-"机构认证"进入机构认证页面；

3.选择"学校认证"或"机构认证"。学校、机构认证只能选择认证一项，认证支付前请首先确认认证类别，一旦认证通过，不可更改。提交的资料审核通过后，需要在线交纳980元机构认证费用，提高机构的可信度。认证完成后会赠送壹客CRM客户关系管理系统一套，提供5端口使用权；

4.如果选择"学校认证"，需提交在有效期内的办学许可证，需保证学校名称等文字及印章清晰可见；

5.如果选择"机构认证"，需提交在有效期内的营业许可证，需保证企业名称等文字及印章清晰可见。

官网暂不支持机构认证。
