如何发布资料

访问官网www.imgaozhao.com并登录：

1.在左侧导航栏选择"课程管理"-"我的课程"；

2.在右侧"我的课程"页面中选择"资料"，再点击"添加资料"；

3.根据页面提示设置资料的详细信息，包括"标题"、"科目"、"分类"、"现价"、"原价"（可不填）、"语言"、"类型"、"标签"、"主讲"、"课程封面"（可不上传，建议上传）和"课程图文详情"；

4.在页面最下方"资料上传"部分选择资料并上传，支持DOC、PPT、PDF等众多格式。目前每个资料仅支持上传一个文件。用户购买资料后可以直接在AI学中查看；

5.点击页面下方"发布课程"进行发布。


AI学手机客户端暂不支持资料发布。
